Grid System

Starting from full screen, try resizing the window so that it gets smaller and smaller. You will notice that the two paragraphs will shift from a 1:1 ratio to a 2:1 ratio to a 1:2 ratio. The left cell is 6/9/3 and the right cell is 6/3/9. The first number in both denotes the columns that each cell takes up out of 12 since 12 columns is the default and most optimal. So the left cell takes 6/12 and the right cell takes 6/12 when the window is large. The second number denotes what they do when the window is medium sized and the third number denotes what they do when the window is small sized.

Buttons

Coloring

Options: primary, secondary, success, alert, warning

Changes the button's color based on the class of button that is specified.

Sizing

Options: tiny, small, large, expanded

Changes the button's size based on the class of button that is specified.

Others

Options: hollow, clear

Hollow creates a button without a background but with a border. Clear creates a button without a background nor a border. Disabled creates a button that looks like it is disabled; the disabled class must be added to actually disable it.
