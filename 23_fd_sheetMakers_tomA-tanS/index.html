Buttons

Coloring

Options: primary, secondary, success, alert, warning

Changes the button's color based on the class of button that is specified.

Sizing

Options: tiny, small, large, expanded

Changes the button's size based on the class of button that is specified.

Others

Options: hollow, clear

Hollow creates a button without a background but with a border. Clear creates a button without a background nor a border. Disabled creates a button that looks like it is disabled; the disabled class must be added to actually disable it.
